# Śledzenie Szkieletu z użyciem MediaPipe Holistic


## Możliwości modelu
* **21 punktów dłoni**
* **468 punktów twarzy**
* **33 punkty postawy**

## Wydajność i środowisko
* **Wydajność:** 15 - 20 fps na procesorze Intel Core i7-11800H.
* **Środowisko:** Testowane na **Python 3.9**.

## Uruchomienie

1. Ściagnięcie bibliotek
pip install -r requirements.txt

2. Uruchomienie
python medhol.py
